# Downpour releases (macOS)

Download the latest installer:

**[Downpour.dmg](Downpour.dmg)** — macOS installer (Safari app + Chrome extension)

Versioned builds are also kept here as `Downpour-<version>.dmg`.

## Install

1. Open the DMG
2. Double-click **Install Downpour.command**
3. Choose **Safari**, **Chrome**, or **Both**
4. Follow the on-screen prompts

**Safari:** enable **Downpour** in **Safari → Settings → Extensions**

**Chrome:** load the unpacked extension from the folder the installer opens in Finder (`chrome://extensions` → Developer mode → Load unpacked)

See the main [README](../README.md) for details.
