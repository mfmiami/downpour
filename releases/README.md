# Downpour releases (macOS)

Download the latest installer:

**[Downpour.dmg](Downpour.dmg)** — macOS Safari extension (Apple Silicon)

Versioned builds are also kept here as `Downpour-<version>.dmg`.

## Install

1. Open the DMG
2. Double-click **Install Downpour.command**
3. Enable **Downpour** in **Safari → Settings → Extensions**

See the main [README](../README.md) for details.
